= Nested todo/scrum-ish plugin for redmine

* Supporting redmine 2.1

= Working features

* new todo
* delete todo
* edit todo
* toggle todo completion status
* supports self todos and their CRUD operations

= Installation

* Checkout the repository
* Copy folder into plugins folder of redmine
* Rename folder to "redmine_todo_scrum_plugin"
* Run command "rake redmine:plugins:migrate"
* Restart the server

You are ready to go
